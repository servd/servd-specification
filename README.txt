ServD-specification

This is the location of the core elements of the servd-specifation

Programable components will be included here for reference that are aligned with the specification as published.

ServD_model.xmi		This is the Current Standard compliant ServD XMI model
Clean_EA_XMI.xslt	This is a stylesheet that was used to "clean" the EA Exported XMI file
Validate_EA_XMI.xslt	This stylesheet prepares a HTML page with some validation of the XMI
			File, and then a summary of the operations, enumerations, packages
			and classes in the Model (complete with their cardinality)
